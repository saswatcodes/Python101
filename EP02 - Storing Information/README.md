# Storing Information

### 1. Integers
- Represented as `int`.
- Can be stored between: **-9,223,372,036,854,775,808** and **9,223,372,036,854,775,808**
- Numeric Bases:
  - `bin()`: Base 2 : 0 and 1
  - `oct()`: Base 8 : 0 through 7, & : Base 10 : The usual number system
  - `hex()`: Base 16 : 0 through 9 , A through F
